Developing Data Products - Course Project

Visualizing Trends in Bonds Issuances in East Asian Countries

Justin Wong

What does the Application do?

  1. Draws data from AsianBondsOnline.
  2. Cleans the data.
  3. Formats the data and plots a neat graph based on the user's inputs.

What does the user need to do?

  1. Simply select the Country, Entity type and Currency of choice.
  2. Enter submit.
  3. Viola, the application will draw the data and plot automatically.
